Case study

Building my Amazon clone

An ecommerce clone is useful because it contains many real interface patterns: navigation, cards, search, product grids, ratings, prices, and responsive behavior.

Layout lessons

I practiced spacing, grid layout, reusable cards, and making text readable on different screen sizes. The goal was not to copy blindly, but to understand why the interface is structured that way.

Next steps

The project can grow with a cart system, product filtering, saved cart data, and checkout form validation.
