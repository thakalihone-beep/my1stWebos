Project story

My first WebOS project

The first version of my WebOS project was simple: icons, windows, a calculator, notes, and a taskbar. Even with a small feature set, it taught me how much state a user interface has to remember.

What I learned

I learned how to show and hide app windows, update the clock with JavaScript, and connect buttons to page behavior. The biggest lesson was that even beginner projects should be organized clearly.

What I would improve

The next version should use reusable app components, better keyboard support, saved notes, and responsive layouts so the experience works beyond desktop screens.
